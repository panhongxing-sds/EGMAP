# Official MASPO Base

This repository was initialized by cloning the official MASPO code:

- Remote: https://github.com/wangzx1219/MASPO.git
- Base commit: e79aa8e

The implementation keeps MASPO's prompt optimization and topology machinery as the base, then adds Experience-Guided Multi-Agent Prompting (ExHandoff): memory retrieval, adaptive handoff, disagreement verification, and residual selection.
